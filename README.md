#  Sequelize ORM Project (Express + PostgreSQL)

This is a simple backend API built using Express.js and Sequelize ORM with PostgreSQL database.  
It demonstrates basic CRUD operations and model relationships.

---

## Technologies Used
- Node.js
- Express.js
- Sequelize ORM
- PostgreSQL
- dotenv
